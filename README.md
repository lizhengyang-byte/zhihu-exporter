# 知乎内容批量导出工具

一组油猴（Tampermonkey）脚本，用于将知乎内容导出为 Markdown 文件，方便本地存档或在 Obsidian 等工具中阅读。

## 脚本列表

### 1. 答主/问题内容导出 (`main.js`)

导出知乎答主主页的全部内容，或某个问题下的所有回答。

**功能特点：**
- 导出答主的回答、文章、想法
- 导出一个问题下的所有回答
- 支持按热度或时间排序
- 生成带 Frontmatter 和目录的合集 Markdown 文件
- 支持 Obsidian 双向链接和标准 Markdown 两种链接风格

### 2. 收藏夹批量导出 (`zhihu-collections-export.user.js`)

导出知乎收藏夹中的文章，每篇文章保存为独立的 Markdown 文件，支持并行下载。

**功能特点：**
- 列出用户所有收藏夹，支持勾选指定收藏夹或一键全部导出
- 每篇文章独立为 `.md` 文件，支持并行下载（默认 6 路并发）
- 文件组织方式可选：平铺输出或按收藏夹名分类前缀
- 自动补充文章正文（通过知乎 API 获取完整内容）
- 支持 Obsidian 友好格式（Frontmatter + 链接）
- 实时进度显示，支持取消操作
- 自动处理文件名冲突和非法字符

## 安装方法

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 打开扩展的管理面板，新建脚本
3. 将对应 `.js` 文件的内容粘贴进去，保存
4. 打开知乎相关页面即可看到浮动面板

## 使用说明

### 收藏夹导出

1. 打开任意知乎用户主页（`https://www.zhihu.com/people/*`）
2. 页面右上角会出现橙色 **📚** 浮动按钮
3. 收藏夹列表加载后，勾选要导出的收藏夹
4. 选择文件组织方式和链接风格
5. 点击「开始导出」，脚本会自动获取内容并并行下载

### 答主/问题导出

- 在答主主页或问题页面，点击浮动面板选择要导出的内容类型
- 生成一个包含所有内容的合集 Markdown 文件

## 输出格式

每篇文章的 Markdown 文件包含：

```yaml
---
title: "文章标题"
author: "作者名"
source: "原文链接"
collection: "所属收藏夹"
export_date: "导出时间"
created: "创建日期"
votes: 123
comments: 45
tags:
  - 知乎导出
  - 收藏夹名称
---

# 文章标题

> **作者**: 作者名 · 📅 创建日期 · 👍 123 · 💬 45
> **来源**: [文章标题](原文链接)
> **收藏夹**: 收藏夹名称

--- 正文内容 ---
```

## 技术细节

- 使用知乎开放的 REST API 获取数据
- HTML 内容通过 DOM 解析器转换为 Markdown
- 采用 Worker 池模式实现并发下载，兼顾性能与浏览器限制
- 自动处理 API 限流（429 状态码重试）
- 无需任何外部依赖
